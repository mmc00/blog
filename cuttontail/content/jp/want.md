## 「意向形」的变形规则（以ます形为变形基准）

1. 一类动词：`い段` → `お段` + `う`　(おう读成长音)
	| ます形 |  | 意向形 |
	| --- | --- | --- |
	| かきます | → | かこう |
2. 二类动词：去掉 `ます` + `よう`
	| ます形 |  | 意向形 |
	| --- | --- | --- |
	| 食べます | → | 食べよう |
3. 不规则动词
	| ます形 |  | 意向形 |
	| --- | --- | --- |
	| します | → | しよう |
	| きます | → | こよう |
